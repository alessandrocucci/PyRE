REST API, prototipazione rapida
con Flask e SqlAlchemy

Mercoledí 11 Gennaio 2017 h19:00

Impact HUB Reggio Emilia

Sommario

Per il primo incontro del PyRE si é scelto un argomento che negli ultimi anni ha avuto ampia diffusione in campo IT: i web service di tipo REST. Nel corso della serata verrá illustrato un esempio pratico di come iniziare lo sviluppo di un'applicazione REST lato server, utilizzando strumenti ormai noti nel panorama Python: Flask, un microframework che permette la realizzazione in poche righe di codice di un'app server side, e SqlAlchemy, un ORM che useremo per l'accesso e la manipolazione dei dati usando un linguaggio pythonico, senza la preoccupazione di dover fare delle query custom.

Speaker

Alessandro Cucci, 31 anni, lavora attualmente come Python Developer presso Energee3 srl, azienda di consulenza informatica con sedi a Reggio Emilia, Milano, Torino e Roma. In Energee é Responsabile di Competenza per il linguaggio Python, si occupa della formazione ai nuovi tirocinanti/tesisti/neo-assunti, offre supporto tecnico ai developer Python, gestisce la revisione del codice Python aziendale.
In ambito community, é attivo come Contributor per la Python Software Foundation, nel 2016 ha rilasciato due patch per la release 3.6 di Python.
Tra i suoi interessi, l'utilizzo di Python nel campo dell'IoT, Machine Learning e Big Data.
